How to open project

1. Clone this repo
2. Change Project name to what you want <br />
    -> Folder name <br />
    -> In package.json <br />
    -> In app.json <br />
    -> In src/app.js (default is 'mockreddit') <br />
3. Run command <br />
    -> cd projectName <br />
    -> react-native run-ios <br />
    
Reference: https://facebook.github.io/react-native/docs/getting-started.html
